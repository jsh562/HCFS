# Han Conditional Fair Source v1.0 (HCFS)

The HCFS software license by James Han, published as a readable web page.

**📄 Read the license:** https://jsh562.github.io/HCFS/

## What's here

- [`LICENSE`](LICENSE) — the **canonical, legally binding** license text. This file is the source of truth.
- [`index.html`](index.html) — a styled, generated web rendering of the license for convenient reading (served via GitHub Pages).
- [`build_page.py`](build_page.py) — generator that builds `index.html` from `LICENSE`.

## Keeping the page in sync

`index.html` is generated from `LICENSE`. After editing `LICENSE`, regenerate the page so the two stay identical:

```sh
python build_page.py
```

Then commit both `LICENSE` and `index.html`.
